NAME
====

**Astro::Utils** - Provides utility functions for astronomical calculations

SYNOPSIS
========

```raku
use Astro::Utils;
```

DESCRIPTION
===========

**Astro::Utils** is a collection of utility functions from several popular astronomy-related books by authors such as Meeus and Lawrence.
